# v0.6.1 — Refinery Crafting Calculator Design

**Release**: v0.6.1
**Status**: Proposed
**Date**: 2026-07-09
**Reference**: https://dune.gaming.tools/crafting-calculator

---

## Abstract

A crafting calculator embedded in the Dune Ops Observability addon that computes raw material requirements for refinery processing. Operators and players select a desired output (e.g., "Processed Spice ×100"), and the calculator returns the exact raw material inputs needed with production ratios.

**Scope**: Spice refineries, ore refineries, and chemical refineries only. Full item crafting (weapons, armor, consumables) is out of scope for v0.6.1.

---

## Refinery Types

### Spice Refinery
| Input | Output | Ratio |
|-------|--------|:-----:|
| Raw Spice ×5 | Processed Spice ×1 | 5:1 |
| Raw Spice ×10 | High-Grade Spice ×1 | 10:1 |

### Ore Refinery
| Input | Output | Ratio |
|-------|--------|:-----:|
| Copper Ore ×2 | Copper Ingot ×1 | 2:1 |
| Iron Ore ×2 | Iron Ingot ×1 | 2:1 |
| Azurite Ore ×3 | Azurite Crystal ×1 | 3:1 |

### Chemical Refinery
| Input | Output | Ratio |
|-------|--------|:-----:|
| Raw Oil ×4 | Processed Fuel ×1 | 4:1 |
| Raw Oil ×3 | Lubricant ×1 | 3:1 |
| Sulfur ×2 + Raw Oil ×2 | Explosive Compound ×1 | 2:2:1 |

## Technical Implementation

### Data Source

Recipe ratios are static game data — they don't change per-server. They can be hardcoded in the addon's `data-providers.js` as a structured catalog, same pattern as the existing sample data provider.

```javascript
// data-providers.js
export const REFINERY_RECIPES = {
  spice: {
    "Raw_Spice": {
      "Processed_Spice": { ratio: 5, timePerCycle: 30 },
      "High_Grade_Spice": { ratio: 10, timePerCycle: 60 }
    }
  },
  ore: {
    "Copper_Ore": { "Copper_Ingot": { ratio: 2, timePerCycle: 20 } },
    "Iron_Ore": { "Iron_Ingot": { ratio: 2, timePerCycle: 20 } },
    "Azurite_Ore": { "Azurite_Crystal": { ratio: 3, timePerCycle: 40 } }
  },
  chemical: {
    "Raw_Oil": {
      "Processed_Fuel": { ratio: 4, timePerCycle: 25 },
      "Lubricant": { ratio: 3, timePerCycle: 20 }
    },
    "Sulfur+Raw_Oil": { "Explosive_Compound": { ratio: 1, timePerCycle: 45, inputs: { Sulfur: 2, Raw_Oil: 2 } } }
  }
};
```

### UI Rendering

The calculator is a new panel rendered in the addon's `web/addon.js`, accessible from the Inventory or a new Tools tab. It uses the existing Dune Rise typeface and Arrakis background.

### No Server-Side Changes

No core fork changes needed. The calculator is entirely client-side JavaScript using static recipe data. No bridge actions, no database queries.
