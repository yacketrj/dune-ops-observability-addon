# Governance Templates

This directory contains reusable templates for PRs, releases, security evidence, SBOM evidence, SOC2-style controls, and risk reviews.

Use release-specific templates when they are stricter than these shared templates.

## Required PR Templates

- `pr-body-template.md`
- `pr-validation-template.md`
- `pr-security-output-template.md`
- `pr-risk-review-template.md`

## Required Release Templates

- `release-gate-template.md`
- `release-decision-template.md`

## Required Security and Control Templates

- `shift-left-security-gate-template.md`
- `local-security-tool-output-template.md`
- `sbom-impact-template.md`
- `soc2-style-controls-template.md`
- `risk-acceptance-template.md`

## Rule

Templates are evidence scaffolds. A PR or release is not approved simply because a template exists; the template must be populated with actual output or a documented risk acceptance.
